# Test Plan: Contact & Company

**URL**: [https://www.mijet.eu/gb/contact-us](https://www.mijet.eu/gb/contact-us)

## Test Cases

| ID | Title | Description | Expected Result |
| :--- | :--- | :--- | :--- |
| CT-01 | Contact Form Submission | Fill out all fields correctly and submit the form. | Success message is displayed after submission. |
| CT-02 | Form Validation | Submit form with missing or invalid fields (e.g., mail). | Error messages appear highlighting the issue. |
| CT-03 | Google Maps | Interact with the embedded Google Map. | Map is interactive, shows the correct location. |
| CT-04 | Direct Contact Links | Click on phone numbers and email links. | Triggers the default calling/emailing application. |
| CT-05 | About Us Content | Read and verify the company's informational content. | Information is accurate and consistent with the brand. |
| CT-06 | Legal/Privacy Links | Click on "Legal Notice" or "Privacy Policy" links. | These pages load correctly with the relevant legal text. |
| CT-07 | Language Switch | Change language using the selector. | Form labels, info text, and page content translate correctly. |
